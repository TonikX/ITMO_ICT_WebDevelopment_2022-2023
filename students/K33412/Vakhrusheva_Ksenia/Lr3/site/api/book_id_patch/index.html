    Изменить информацию о книге по ID

    Описание

    url: /book/{id}/
method: PATCH
auth: Token, same Editor profile

    Пример запроса

    {
    "authors": [
        {
            "user": 12
        },
        {
            "user": 1
        }
    ],
    "name": "My New Book"
}
