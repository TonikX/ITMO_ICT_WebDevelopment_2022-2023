# Навыки

Навыки воинов.

## Показать все навыки

Выводит информацию обо всех навыках

**URL** : `/war/skills/`

**Method** : `GET`

### Success Responses

**Code** : `200 OK`

**Content** : `{[]}`

```json
{
    "Skills": [
        {
            "id": 1,
            "name": "skill"
        }
    ]
}
```
