# Show, delete and replace an expert

Shows info about an expert

**URL** : `/experts/<id>`

**Method** : `GET PUT PATCH`

**Auth required** : NO

**Permissions required** : None

**Data constraints** : `{}`

## Success Responses

**Code** : `200 OK`

**Content** : `{}`

```json
{
    "id": 1,
    "name": "Juan",
    "last_name": "Andreev",
    "club": "Hunt",
    "ring_field": 1
}```
