# Show breeds of members of a specified club

Shows breeds of members of a specified club

**URL** : `/club_breeds/<id>`

**Method** : `GET`

**Auth required** : NO

**Permissions required** : None

**Data constraints** : `{}`

## Success Responses

**Code** : `200 OK`

**Content** : `{}`

```json
{
    "name": "Royal",
    "members": [
        {
            "breed": "r"
        }
    ]
}}```
