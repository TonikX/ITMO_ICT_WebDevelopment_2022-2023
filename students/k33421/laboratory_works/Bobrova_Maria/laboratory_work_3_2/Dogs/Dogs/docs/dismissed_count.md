# Shows number of dismissed participants for the specified year

Show number of dismissed participants for the specified year

**URL** : `/dismissed_count/id`

**Method** : `GET`

**Auth required** : NO

**Permissions required** : None

**Data constraints** : `{}`

## Success Responses

**Code** : `200 OK`

**Content** : `{}`

```json
{
    "year": 2021,
    "dismissed_count": 2
}
```
