## /store/users
Получить всю информацию о пользователе

**URL** : `/LIBRARY/users`

**Method** : `GET`

**Auth required** : YES

**Permissions required** : None

**Code** : `200 OK`

```json
[
    {
        "username": "username",
        "first_name": "first name",
        "last_name": "last name",
        "tel": 123456
    },
]
```
