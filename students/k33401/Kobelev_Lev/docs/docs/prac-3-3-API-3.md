# Get all jobs
Выводит информацию обо всех профессиях.

## Information

**URL:** `/war/jobs/`

**Method:** `GET`

**AUTH required:** NO

**Permissions required:** None

**Data constraints:** `{}`

## Success Responses

**Code:** `200 OK`

**Content:** `{[]}`

        "title": "Game Designer",
