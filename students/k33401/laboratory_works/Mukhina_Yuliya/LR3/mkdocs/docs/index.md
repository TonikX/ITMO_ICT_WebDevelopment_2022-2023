# Мухина_Lr3
# REST API погоды



### Документация API
`/api/cities` (GET) - список городов

Пример конкретного города:
```json
{
  "id": 1,
  "name": "Санкт-Петербург",
  "description": "Город на Неве",
  "lat": 59.9375,
  "lon": 30.308611,
  "country": "ru"
}
```
`/api/countries` (GET) - список стран + городов в них

Пример конкретной страны:
```json
{
  "code": "us",
  "cities": [
    {
      "id": 3,
      "name": "Нью-Йорк",
      "description": "Большое яблоко",
      "lat": 40.73061,
      "lon": -73.935242,
      "country": "us"
    },
    {
      "id": 4,
      "name": "Лос-Анджелес",
      "description": "Там находится Голливуд",
      "lat": 34.052235,
      "lon": -118.243683,
      "country": "us"
    }
  ],
  "name": "США"
}
```
