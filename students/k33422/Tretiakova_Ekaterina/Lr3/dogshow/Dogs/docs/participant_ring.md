# Show which rings the participant was on

Shows which rings the participant was on

**URL** : `/participant_ring/<id>`

**Method** : `GET`

**Auth required** : NO

**Permissions required** : None

**Data constraints** : `{}`

## Success Responses

**Code** : `200 OK`

**Content** : `{}`

```json
{
    "rings": [
        1
    ]
}```
