## /store/users
Просмотр списка пользователей

**URL** : `/store/users`

**Method** : `GET`

**Auth required** : YES

**Permissions required** : None

**Code** : `200 OK`

```json
[
    {
        "username": "wan",
        "first_name": "wan",
        "last_name": "1",
        "tel": null
    },
    {
        "username": "user1",
        "first_name": "f1",
        "last_name": "l1",
        "tel": null
    }
]
```
